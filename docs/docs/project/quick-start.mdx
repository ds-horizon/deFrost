### Installation

1. Add `de-frost` to project.

<Tabs groupId="package-manager">
  <TabItem value="yarn" label="yarn">
    ```sh
    yarn add @d11/de-frost
    ```
  </TabItem>
  <TabItem value="npm" label="npm">
    ```sh
    npm install @d11/de-frost
    ```
  </TabItem>
</Tabs>

2. Prepare Your Android Device - Follow the steps outlined in the [Setup Guide](/docs/project/setup) to enable USB Debugging, Profile HWUI Rendering, and verify ADB connection.

3. You start by generating an APK that has React commit tracking code added:

<Tabs groupId="package-manager">
  <TabItem value="yarn" label="yarn">
    ```sh
    yarn de-frost create-build -f Staging -v Release
    ```
  </TabItem>
  <TabItem value="npm" label="npm">
    ```sh
    npx de-frost create-build -f Staging -v Release
    ```
  </TabItem>
</Tabs>

Here -f is an optional flag.

4. Once the APK is ready, installed on your device and all permissions given, you can begin recording frame and React commit data

<Tabs groupId="package-manager">
  <TabItem value="yarn" label="yarn">
    ```sh
    yarn de-frost record -p com.example.myapp
    ```
  </TabItem>
  <TabItem value="npm" label="npm">
    ```sh
    npx de-frost record -p com.example.myapp
    ```
  </TabItem>
</Tabs>

This will start recording and will dump the data into the `data` folder.

5. Finally, you can open the web dashboard to analyze the data and identify any potential frozen frames.

<Tabs groupId="package-manager">
  <TabItem value="yarn" label="yarn">
    ```sh
    yarn de-frost show-dashboard -d ./data
    ```
  </TabItem>
  <TabItem value="npm" label="npm">
    ```sh
    npx de-frost show-dashboard -d ./data
    ```
  </TabItem>
</Tabs>
